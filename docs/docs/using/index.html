Using #

pyControl #

This nosepoke works with the pyControl behavior framework. Specifically, this means that the RJ45 pinout is compatible with pyControl. Using an ethernet cable, you can connect the nosepoke to an original breakout board or a D-series breakout board.

The latest sofware release can be downloaded here. Additional pyControl harware can be purchased from the Open Ephys Store.

Example task #

Instuctions for using the pyControl GUI to upload and run a task can be found here.

Below is an example task you can run to test the nosepoke. This task assumes there is a nosepoke connected to port 1 of your breakout board and the hardware_definition.py file (included with the the software download) has been uploaded to your pyBoard. Once the task is running, the nosepoke LED will turn on whenever a poke is being detected.

from pyControl.utility import *
from devices import *
import pyControl.hardware as _h

## uncomment one of the following two lines depending out which breakout board you are using
board = Breakout_1_2()  # original pyboard breakout
# board = Breakout_dseries_1_6()  # d-series pyboard breakout

nose = Nosepoke(board.port_1, nose_event="poke")

states = ['init_state']
events = ['poke', 'poke_out']

initial_state = 'init_state'

def init_state(event):
    if event == 'poke':
        nose.LED.on()
    elif event == 'poke_out':
        nose.LED.off()

For more information on writing your own tasks, refer to the pyControl documentation.
